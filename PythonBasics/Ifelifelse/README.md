## Conditional statements are if, else, elif statements. 

### Basics: If, Elif, Else Follow the format 

if condition: 
  expression 
elif: 
  expression
else: 
  expression 

Incoporating "And, Or, Not" to conditional statements can add dimensions by incoporating And, Or, Not parameters.
